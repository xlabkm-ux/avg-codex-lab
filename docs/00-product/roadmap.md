# Roadmap

## Phase 0 — Operating System

Outcome: Codex agents can safely work in the repo.

## Phase 1 — Core Dialogue

Outcome: user can create a project and receive structured AVG responses.

## Phase 2 — Validation Engine

Outcome: AVG can extract claims, classify them, flag risks, and repair weak statements.

## Phase 3 — Visual Map

Outcome: user can see and edit a concept map.

## Phase 4 — Retrieval and Documents

Outcome: AVG uses user documents and project memory safely.

## Phase 5 — Voice and Collaborative Sessions

Outcome: AVG becomes a live creative thinking environment.
