# Interaction Model

AVG UI has four main surfaces:

1. Dialogue — the live conversation.
2. Map — the structured concept graph.
3. Validator — claims, risks, repairs.
4. Artifact — polished output.

## User Flow

```text
raw idea -> creative variants -> map -> validation -> artifact -> next action
```

## UX Rule

Never make validation feel like punishment. It should feel like sharpening.
