# Design System

## Visual Mood

- calm;
- precise;
- intelligent;
- low-noise;
- spacious;
- map-oriented.

## Core Components

- ChatPanel;
- ModeSwitcher;
- ConceptMap;
- ClaimInspector;
- RiskBadge;
- ArtifactEditor;
- SourcePanel;
- VoiceSessionPanel;
- EvaluationBadge;
- MapDiffViewer.

## Risk Badges

Risk badges should be visible but not alarming.

Examples:

- Strong word;
- Metaphor only;
- Hypothesis;
- Needs source;
- Level mixing;
- Map/territory risk.
